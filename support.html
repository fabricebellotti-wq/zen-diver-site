Avant d'écrire, quelques infos utiles

Je ne reçois pas mon code de connexion

  • Vérifie tes spams / courriers indésirables.
  • Le code peut prendre 1-2 minutes à arriver selon ton fournisseur d'email.
  • Vérifie que tu as tapé la bonne adresse — sinon ré-essaye avec « Renvoyer le code ».
  • Si rien après 5 minutes, contacte-nous avec ton adresse email exacte.

Le déverrouillage biométrique ne fonctionne pas

  • Vérifie que la biométrie est activée dans les réglages de ton appareil (Réglages iOS → Face ID / Touch ID et code sur iPhone, Paramètres → Sécurité / Biométrie sur Android).
  • Dans l'app, va dans Mon profil → Mon compte et active le déverrouillage biométrique.
  • Si l'app n'a jamais demandé l'autorisation, supprime-la et réinstalle-la.

J'ai acheté Pro mais je suis toujours en version gratuite

  • Va dans Mon profil → Restaurer mes achats (depuis l'écran « Passer Pro »).
  • Vérifie que tu es connecté avec le même compte que celui qui a effectué l'achat (Apple ID sur iPhone, compte Google sur Android).
  • Si le problème persiste, contacte-nous avec ton numéro de transaction : sur iPhone via Réglages iOS → [ton nom] → Abonnements / Achats ; sur Android via Google Play → Paiements et abonnements → Budget et historique.

Je veux supprimer mon compte

  • Va dans Mon profil → Mon compte → Supprimer mon compte.
  • L'action est définitive : toutes tes plongées, brevets et documents sont effacés.
  • Si tu veux récupérer tes données avant suppression, contacte-nous pour un export.
